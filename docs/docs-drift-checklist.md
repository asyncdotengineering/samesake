# Public Docs Drift Checklist

Use this checklist before changing README, docs, examples, or package READMEs.

- The first screen says Samesake, not deprecated project names.
- The page identifies the category: TypeScript-first visual-commerce search engine compiler, starting with fashion.
- The page links to a runnable example or clearly states when an external dataset is required.
- Any proof claim links to [Fashion Search Proof](./fashion-search-proof.md) or a specific result artifact.
- Config examples use `samesake.config.ts` and current SDK exports.
- Fashion search language distinguishes multi-space retrieval from keyword-only, vector-only, and flat visual similarity.
- Infrastructure details are accurate but do not replace the product framing.
- Archived docs remain under `docs/archive/` and are not linked as current guidance.
